# Starts wireshark for pod on kubernetes reachable via teleport

## Installation
pip3.7 install --user -r requirements.txt

## Usage
usage: main.py [-h] [--pod POD] [--container CONTAINER]
               [--interface INTERFACE]

optional arguments:
  -h, --help            show this help message and exit
  --pod POD
  --container CONTAINER
  --interface INTERFACE
